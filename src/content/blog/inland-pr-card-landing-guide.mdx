获得加拿大永久居留权（PR）并完成首次登陆，是移民旅程中最重要的里程碑。从拿着 CoPR 登陆纸踏上加拿大土地的第一天起，了解居住义务、PR 卡申请、旅行权利及入籍要求，将帮助您稳固并发展您的永久居民身份。

## 首次登陆流程（CoPR 登陆纸）

获批永久居留权后，您将收到入境口岸信件或电子版确认入境许可，即 **CoPR（Confirmation of Permanent Residence，永久居民确认书）**。首次登陆时：

1. **持 CoPR 和有效护照入境**：在加拿大任意口岸（机场/陆路/海路）向边境官员（CBSA）申报 PR 登陆意向。
2. **接受边境官员面谈**：官员核查 CoPR 信息与护照，确认您的身份。如有随行家庭成员，各人均需出示各自的 CoPR。
3. **签署 CoPR 文件**：边境官员会在您的 CoPR 上盖章并签署，这是您成为加拿大永久居民的法律确认时刻。
4. **保存登陆印章**：确保护照上有登陆日期印章，CoPR 原件妥善保管。登陆日期是计算居住义务和入籍资格的起始点。
5. **申请 SIN 和省级服务**：登陆后尽快前往 Service Canada 申请 SIN，到省级政府注册医疗卡，并办理驾照换领。

> 登陆后通常不会立即收到 PR 卡——PR 卡会在您注册地址后约 **42–56 天**邮寄到达。登陆时请确认加拿大境内邮寄地址正确。

## PR 卡申请与有效期

首次登陆后，IRCC 会根据您在 CoPR 上填写的加拿大地址自动启动 PR 卡制作并邮寄，通常无需单独申请：

- PR 卡有效期：通常为 **5 年**（部分特殊情况为 1 年）
- 首张 PR 卡自动制作，无需缴费（后续续签费 $50 CAD）
- 收到 PR 卡时间：约 42–56 天（从登陆日起算）
- PR 卡仅寄往加拿大境内地址，无法寄往境外
- 如地址变更，需在 IRCC 账号中更新，或致电 IRCC 客服（1-888-242-2100）更新邮寄地址

## 居住义务（Residency Obligation）

加拿大永久居民有维持居住义务的法律要求：

- 要求：在**任意连续 5 年内**，在加拿大实际居住至少 **730 天**（约 2 年）
- 计算方式：**滚动计算**——IRCC 以申请日向前追溯 5 年的天数
- 未满足居住义务可能导致 PR 身份被终止（需经过听证程序）

**可计入居住义务的情况：**
- 陪同加拿大公民配偶/父母在境外居住
- 受雇于加拿大联邦/省政府或加拿大公司在境外工作（需提供证明）

> 730 天是最低要求，而非"安全线"。如果您的居住天数接近 730 天，续签 PR 卡时会面临严格审查。建议保持详细的出入境记录（护照印章、机票、住宿记录）以备 IRCC 核查。

## 境外旅行：PR 旅行证件（PRTD）

PR 卡过期或遗失时在境外，无法持过期 PR 卡搭乘国际航班返加，需申请 PRTD：

- PRTD（Permanent Resident Travel Document）在加拿大驻外使馆/高专署申请
- 申请费：$50 CAD
- 处理时间：通常 2–4 周（紧急情况可申请加急）
- 需提供：护照、CoPR 副本、PR 卡（如已过期附过期卡）、符合居住义务的证明材料
- PRTD 为**单次使用**，持 PRTD 入境后，在加拿大境内申请 PR 卡续签

## 入籍要求（成为加拿大公民）

| 要求 | 具体标准 |
|------|----------|
| 居住要求 | 5 年内在加实住 **≥ 1095 天**（3 年），PR 前以临时居民身份在加居住可折半计入（最多 365 天） |
| 语言要求 | 英语或法语 CLB 4 级或以上（18–54 岁申请人） |
| 报税要求 | 5 年内至少 3 年在加拿大报税（如有纳税义务） |
| 公民知识测试 | 18–54 岁须通过公民知识测试（基于《发现加拿大》手册） |
| 宣誓 | 参加公民宣誓典礼，宣誓效忠加拿大（18 岁以上） |
| 入籍申请费 | $630 CAD（成人），$100 CAD（18 岁以下） |

> 入籍申请处理时间约 12–24 个月。建议在居住天数刚满足要求后即提交申请，以节省等待时间。

## 登陆后必办 6 件事

1. **申请 SIN**：前往 Service Canada，持 CoPR + 护照当天即可获得。
2. **注册省级医疗保险**：各省等待期通常 3 个月，尽早注册。期间购买私人医疗保险过渡。
3. **开设加拿大银行账户**：持 CoPR + 护照 + SIN 可在各大银行开设账户。
4. **办理驾照**：持中国驾照（附公证翻译）通常可豁免部分考试，直接换领省级驾照（各省规定不同）。
5. **更新/确认地址**：在 IRCC 账号中确认 PR 卡邮寄地址正确，并注册 CRA 账号用于报税。
6. **建立信用记录**：申请信用卡（初始额度可能较低），按时还款，为未来租房、购车、贷款做准备。

## 常见问题

**获得 PR 后可以继续持临时工签工作吗？**

一旦完成登陆（签署 CoPR），即成为永久居民，原临时工签失效。登陆后，您的工作授权来自 PR 身份本身，无需持有任何工作许可，可在加拿大任意合法就业。

**居住义务中的 730 天是连续的还是累计的？**

是累计的，不需要连续。730 天是在任意连续 5 年内的累计在加拿大居住天数。记录每次出入境的日期非常重要——建议使用电子表格持续跟踪您的出入境记录。

**PR 身份可以被撤销吗？**

可以，但需经过正式法律程序。PR 身份可能因以下原因被终止：未满足居住义务（需经过移民上诉庭听证）、刑事定罪（严重罪行）、移民欺诈等。居住义务未达标**不会自动失去 PR**——需 IRCC 启动调查程序，申请人有权申诉。

**我的孩子在获得 PR 后在加拿大出生，是加拿大公民吗？**

是的。在加拿大出生的孩子自动获得加拿大公民身份（出生公民权），无论父母是什么身份。出生后需申请加拿大出生证明（Birth Certificate）及加拿大护照。父母的 PR 身份不影响孩子的公民身份。
