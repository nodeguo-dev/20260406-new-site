在加拿大定居的永久居民需要定期更新多项证件，并及时向相关机构报告地址、护照等信息变更。本文整理了最常用的证件更新流程、IRCC 表格清单及关键联系信息。

## PR 卡续签（IMM 5444）

PR 卡有效期通常为 5 年。在 PR 卡到期前至少 **6 个月**申请续签：

- **申请表格**：IMM 5444（可在 IRCC 官网下载）
- **申请费**：$50 CAD
- **申请方式**：在线（IRCC 账号）或纸质邮寄
- **处理时间**：约 **102 天**（截至 2025 年）
- 申请期间 PR 卡过期，可用 PRTD（境外旅行）或等待新卡到达（境内无需担心）

**续签所需材料：**
- 当前 PR 卡（原件）
- 有效护照（需超过新 PR 卡申请的有效期）
- 近期照片（符合 IRCC 照片规格）
- 证明居住义务满足的文件（出入境记录、房租/纳税记录等）

> 如 PR 卡在境外丢失或过期，无法乘坐国际航班返加，需在就近加拿大使馆申请 **PRTD（PR 旅行证件）**，申请费同为 $50 CAD。

## 换护照后必须同步更新的事项

护照到期换发后，许多永久居民忘记同步更新相关证件，可能导致出入境或申请时出现麻烦：

| 更新事项 | 更新渠道 | 备注 |
|----------|----------|------|
| IRCC 账号中的护照信息 | My IRCC 账号在线更新 | 影响 PR 卡续签和入籍申请 |
| 驾照上的姓名/信息 | 省级 DMV/ServiceOntario 等 | 各省要求不同 |
| 银行账户信息 | 各银行柜台或网银 | 部分银行需要到柜台更新 |
| 税务局（CRA）信息 | My Account 在线更新 | 确保退税等事项准确 |
| 雇主人事档案 | 告知雇主 HR 更新记录 | 影响 T4 报税表和工资单 |

## SIN（社会保险号）更新

SIN 本身通常不需要更换，但以下情况需联系 Service Canada：

- 身份变更（如临时 SIN 转为永久 SIN）：持 CoPR 或 PR 卡到 Service Canada 柜台更新
- 姓名更改（结婚/离婚后）：需提供新护照和法律文件
- SIN 卡丢失：仅用于申请记录核对，不补发实体卡（已停发），可申请 SIN 确认函

> 临时 SIN 以 **9** 开头，是有有效期的。获得 PR 后务必携带 CoPR 前往 Service Canada 更换为永久 SIN，以免影响报税、福利申请和雇佣记录。

## 通知地址变更

地址变更需通知以下机构，否则可能错过 PR 卡邮寄、税务通知或福利发放：

1. **IRCC**：通过 My IRCC 账号更新，或致电 **1-888-242-2100**
2. **CRA（税务局）**：通过 My Account 在线更新，或报税时填写新地址
3. **Service Canada（EI/CPP/OAS）**：通过 My Service Canada Account 更新
4. **省级医疗（OHIP/MSP 等）**：联系所在省的医疗服务机构
5. **银行/金融机构**：各自银行更新

## 常用 IRCC 表格清单

| 表格编号 | 用途 |
|----------|------|
| IMM 5444 | PR 卡续签申请 |
| IMM 5406 | 附加家庭信息 |
| IMM 5409 | 法定声明——附加家庭成员 |
| IMM 5476 | 使用代理人或顾问声明 |
| IMM 5562 | 补充信息 |
| IMM 1294 | 工作许可申请（境内） |
| IMM 5710 | 工作许可延期/恢复 |
| IMM 1249 | 配偶/同居伴侣开放工签 |
| IMM 0008 | 移民申请（主表） |
| IMM 5540 | 资金来源声明 |

> 所有表格可在 [IRCC 官网](https://www.canada.ca/en/immigration-refugees-citizenship/services/application/application-forms-guides.html) 免费下载最新版本。务必使用最新版表格，旧版可能被退件。

## 常见问题

**PR 卡过期了还能在加拿大境内生活工作吗？**

可以。PR 身份本身不因 PR 卡过期而失效。PR 卡只是旅行证件和身份证明，过期后仍可在加拿大境内合法居住和工作。主要影响是：无法用过期 PR 卡搭乘国际航班返加，需及时续签。

**申请 PR 卡续签时，IRCC 会审查我的居住义务吗？**

会。IRCC 在续签审核时可能核查您过去 5 年的在加居住天数。如未满足 730 天的居住义务，续签可能被拒，甚至触发 PR 身份检查。建议保持完整的出入境记录（护照印章、机票、酒店记录）以备核查。

**护照过期但 PR 卡仍有效，需要更新什么？**

PR 卡与护照是独立证件，护照过期不影响 PR 卡有效性，但您需要：① 更新护照；② 在 IRCC 账号中更新新护照信息；③ 入境时同时出示 PR 卡和新有效护照。不需要重新申请 PR 卡。
