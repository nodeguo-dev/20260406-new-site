## 服务目标

对在加拿大境内的申请人来说，"保持身份合规"是所有移民规划的前提。身份出现空白不仅影响工作和生活，还会对未来 PR 申请产生负面影响。

## 主要服务场景

### 场景 1：学签续签
- 在读证明获取指导
- 续签申请时机建议（建议到期前 3-4 个月）
- 境内递交与跟进

### 场景 2：工签续签
- 原雇主继续工作的续签
- 换雇主时的新工签申请（LMIA 或豁免）
- PGWP 转工签的路径规划

### 场景 3：访客延期
- 访问签证延期申请（Visitor Record）
- 境内转学签/工签的时机与条件判断

### 场景 4：桥接开放工签（BOWP）
- BOWP 申请条件核实
- 申请时机与材料准备
- 递交后状态跟踪

### 场景 5：身份衔接规划
- 学签到期前的工签/PR 路径规划
- 避免身份空白的时间线设计
- 多证件到期日管理

## 关键时间节点

| 身份 | 建议开始续签准备 |
|------|-----------------|
| 学签 | 到期前 3-4 个月 |
| 工签 | 到期前 3-4 个月 |
| 访客签证延期 | 到期前 4-6 周 |
| PR 卡续签 | 到期前 6-9 个月 |

## 适合人群

- 学签或工签即将到期，不确定如何续签
- 换工作或换学校，需要了解身份影响
- PR 申请已递交，工签快到期，需要 BOWP
- 想在到期前系统梳理身份时间线
