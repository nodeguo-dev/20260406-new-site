## 中学与中学后路径

**中学（Grade 9-12）：**
- 公立学区收费通常 $14,000-$16,000/年（国际生）
- 完成 OSSD（安省高中文凭）后可直接申请大学或 College
- 热门学区：York Region、Peel Region、TDSB（多伦多）

**升学路径参考：**
- 高中 → 大学（需要 OSSD + 语言成绩）
- 高中 → College（门槛相对低，实用性强）
- College → 大学 Pathway（部分 College 与大学有衔接协议）

## 安省 PGWP + 移民衔接建议

```
学签（2年） → 毕业 → PGWP（2-3年）→ 工作经验12个月 → CEC / OINP 申请 PR
```

**关键节点：**
- 选校时确认 PGWP 资格（DLI 编号核查）
- PGWP 在毕业后 180 天内申请（只有一次机会）
- 工签期间同步准备语言成绩和 ECA
